# Sistem Akuntansi Emen Pro

Aplikasi sistem akuntansi untuk pencatatan keuangan dan kegiatan instansi.

## Status: 🚧 Under Development
Aplikasi ini masih dalam tahap pengembangan aktif. Fitur-fitur utama sedang ditingkatkan untuk efisiensi dan pengalaman pengguna yang lebih baik.

## Fitur Utama
- Pencatatan Keuangan (Pengadaan Barang/Jasa)
- Pencatatan Kegiatan
- Dashboard Buku Besar Otomatis (Harian, Mingguan, Bulanan, Tahunan)
